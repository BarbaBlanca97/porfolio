Acerca del proyecto

Es un pequeño plugin para Visual Studio Code que cuenta cuanto tiempo llevas trabajando en un proyecto.

Puede decirte cuanto vas trabajando en la sesión actual (desde la última vez que abriste VS Code) y el tiempo total (desde que se abrió por primera ver el workspace con VS Code).

Lenguajes:

Typescript
